# Live-Commodities-Data-Project
Description
This project fetches **live commodities prices** (Gold, Silver, Crude Oil) using Python and the `yfinance` library. It processes the data, calculates trends, daily % changes, and rolling averages, and visualizes them with Plotly.

Features
- Fetch live commodities prices
- Calculate daily % change and 7-day rolling average
- Generate insights: current trend (uptrend/downtrend)
- Visualize price trends interactively

Technologies
- Python
- Pandas
- Plotly
- yfinance

How to Run: Install required libraries: pip install yfinance pandas plotly
